Writing CSS

The prototype kit uses SCSS syntax of Sass to write CSS.

Sass

Sass is an extension of CSS that gives a load of extra features useful in producing CSS for the mass of different devices and browsers.

You can paste CSS into it without breaking it which is useful for prototyping.

Writing code

You write your Sass in app/assets/sass, and it gets compiled into the CSS used in your page (found in /public/css).

There is already a CSS file included called main.scss which compiles into main.css.

Note that Sass files are identified by the .scss extension.

Each time you make a change in main.scss it will create a new version of main.css. Make sure to write your css in main.scss as anything you put in main.css will get overridden.
