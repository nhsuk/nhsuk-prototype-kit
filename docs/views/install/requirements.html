Requirements

The kit runs on Mac and Windows.

Software you need

You'll need:

  • Visual Studio Code (text editor and terminal)
  • Node.js 8.x.x

You'll usually need admin access to your machine to install the software. If you do not have admin access, ask your IT team to install the software for you.

Visual Studio Code (text editor and terminal)

You need a text editor and terminal to edit and make changes to your prototype.

For a text editor we recommend using Visual Studio Code opens in new window - it's free and has an integrated terminal.

Using the terminal

The terminal lets you type in commands and run programs on your computer.

To open Visual Studio Code's integrated terminal, go to View > Integrated Terminal.

You enter commands in to the terminal like this:

command to be typed in to terminal

Once you’ve typed the command, press enter to send it.

Node.js version 8 LTS

The kit is designed to work with Node.js version 8 LTS. The kit works with any 8.x.x version.

Download and install Node.js

Download version 8 from nodejs.org opens in new window.

Run the installer with all default options.

Once Node is installed

You’ll need to quit and restart the terminal to be able to use Node for the first time.

To check it is installed correctly you can again run:

node --version

If it’s installed correctly it should show a number starting with 8.
